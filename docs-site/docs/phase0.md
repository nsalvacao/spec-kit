# Phase 0: AI System Ideation (Recommended Prerequisite)

Phase 0 provides a structured pre-spec workflow for AI systems:

1. **IDEATE** – Generate a backlog using SCAMPER + HMW  
2. **SELECT** – Score ideas using AI-RICE  
3. **STRUCTURE** – Build the AI Vision Canvas and Vision Brief  
4. **VALIDATE** – Run Gate G0 checks and produce a report

Artifacts are stored under `.spec-kit/`.
